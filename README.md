# Global Superstore Sales Analysis — Power BI

A 2-page interactive Power BI report analyzing sales performance and growth trends across 2011–2014.

## What's Inside
- **Sales Performance Overview** — Average growth by month, YTD, YoY, and 12-month moving average
- **Drill-Through Detail Page** — Sales and profit breakdown by category, sub-category, and region

## Technical Highlights
- Rebuilt flat dataset into a star schema
- Advanced DAX: Sales YTD, Year-over-Year growth, 12-month moving average
- Drill-through navigation from month-level to category/region detail

## Tools
Power BI Desktop · DAX · Star Schema Modeling
